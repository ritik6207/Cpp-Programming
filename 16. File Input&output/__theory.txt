The useful classess for working with files in C++ are:
    1. fstreambase
    2. ifstream  ---> derived form fstreambase
    3. ofstream  ---> derived form fstreambase


In order work with files in C++, you will to open it. Primarily, there are 2 ways to open a file:
    1. using the constructor
    2. using the member function open() of the class
